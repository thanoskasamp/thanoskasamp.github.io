Short Bio

As a WebGIS Developer, I like to use open source technologies to develop WebGIS systems that are capable of handling real-time data from multiple users and APIs.

Following tools are my daily drivers:

  • For mapping library, I use Leaflet.
  • As a frontend language, I use JavaScript and jQuery.
  • I use both Python and PHP as backend languages. Based on purpose, I frequently switch between these two.
  • AJAX for communicating between frontend and backend.
  • For database, I prefer PostgreSQL.
  • I extensively use PostGIS for doing spatial analysis and query.
  • Like to form complex and efficient SQL queries for faster data processing.
  • Proficiency in the Esri ecosystem alongside my dedication to open-source technologies.
